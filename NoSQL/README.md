# MongoDB - Basıcı Tapşırıqlar

Bu layihə MongoDB verilənlər bazası ilə işləmək üçün yazılmış əsas skriptlərdən ibarətdir.

## Skriptlərin Siyahısı

* **0-list_databases**: Sistemdə mövcud olan bütün MongoDB verilənlər bazalarını ölçüləri ilə birlikdə siyahılayır.

## Necə İstifadə Etməli?

Skripti işə salmaq üçün terminalda növbəti əmri yazın:
```bash
